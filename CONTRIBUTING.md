# Contributing notes (tomingtoming fork)

This fork carries the Emscripten/web-port commits for
[ysflight-web](https://github.com/tomingtoming/ysflight-web) as ordinary,
theme-scoped commits on top of upstream
[captainys/YSFLIGHT](https://github.com/captainys/YSFLIGHT).

- **Commits and PRs here are written in English**: every change is meant to
  be cherry-pickable upstream and readable by the upstream / YSCE community.
- Pure bug fixes that apply upstream are kept as **separate commits** so
  they can be proposed as-is.
- Port-specific design records live in the ysflight-web repository
  (`docs/`, written in Japanese).
